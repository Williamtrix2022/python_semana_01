# 🐍 Plan de Aprendizaje: Fundamentos de Python

## Objetivo
Aprender los conceptos fundamentales de Python desde cero para principiantes.

## Contenido Principal

### 1. Configuración Inicial
- Instalar Python
- Usar el intérprete interactivo (REPL)
- Tu primer programa: "Hello, World!"

### 2. Fundamentos Básicos
- **Variables y tipos de datos**: int, float, str, bool
- **Operadores**: aritméticos, comparación, lógicos
- **Input/Output**: entrada del usuario y salida en pantalla
- **Strings**: manipulación básica de texto

### 3. Control de Flujo
- **Condicionales**: if, elif, else
- **Bucles**: for, while
- **Saltos**: break, continue

### 4. Estructuras de Datos
- **Listas**: creación, indexación, métodos
- **Tuplas**: inmutabilidad
- **Diccionarios**: pares clave-valor
- **Conjuntos**: elementos únicos

### 5. Funciones
- Definir funciones (def)
- Parámetros y argumentos
- Return statements
- Scope: variables locales y globales

### 6. Manejo de Errores
- Try, except, finally
- Excepciones comunes

### 7. Temas Avanzados (Introducción)
- Comprensiones de listas
- Importar módulos
- Trabajar con archivos

## Metodología
- Teoría + ejemplos prácticos
- Ejercicios después de cada tema
- Pequeños proyectos integradores
