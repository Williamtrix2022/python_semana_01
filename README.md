# 🐍 Fundamentos de Python - Guía de Aprendizaje

¡Bienvenido! Este es un curso estructurado para aprender Python desde cero.

## 📚 Archivos de Lecciones

| Archivo | Tema | Qué Aprenderás |
|---------|------|---------------|
| `01_inicio.py` | Variables y Tipos | Tu primer programa, tipos básicos (str, int, float, bool) |
| `02_operadores.py` | Operadores | Cálculos, comparaciones, lógica booleana |
| `03_condicionales.py` | Condicionales | if, elif, else, decisiones |
| `04_bucles.py` | Bucles | for, while, break, continue |
| `05_estructuras.py` | Estructuras de Datos | Listas, Tuplas, Diccionarios, Sets |

## 🎯 Cómo Usar Este Material

### Opción 1: Ejecutar archivos desde la terminal
```bash
python 01_inicio.py
python 02_operadores.py
# ... y así sucesivamente
```

### Opción 2: Ejecutar dentro de VS Code
1. Abre el archivo `.py` que quieras
2. Presiona `Ctrl + Shift + P` (o `Cmd + Shift + P` en Mac)
3. Escribe "Run Python File in Terminal"
4. O simplemente presiona el botón ▶️ en la esquina superior derecha

### Opción 3: Usar el Intérprete Interactivo
```bash
python  # Abre el REPL
```
Luego puedes escribir código línea por línea y ver resultados instantáneamente.
